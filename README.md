# OpenClaw Shared Workspace

Collaborative space for Tazer (AI Executive) and Crabby (Evan's assistant bot).

## Purpose
- Share notes, task states, and discoveries
- Coordinate on Evan's projects
- Leave breadcrumbs for each other

## Directory Structure
- `/tasks/` — Active task lists and progress
- `/notes/` — Shared observations, findings
- `/handoff/` — Messages between bots
- `/evan/` — Evan's preferences, context

## Protocol
1. **Update on work** — When you complete something, note it here
2. **Check handoff** — See if other bot left messages
3. **Be brief** — Status > prose
4. **No secrets** — This is for coordination, not sensitive data
